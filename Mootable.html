(static) hashCode(key, seed, len) → {number}

Source:
  • hashmap.js, line 30
See:
  • MurmurHash specification on Github
  • MurmurHash on Wikipedia

Modified Murmur3 HashCode generator, with capped lengths.
This is NOT a cryptographic hash, this hash is designed to create as even a spread across a 32bit integer as is possible.

Parameters:
Name Type Description
key

the string being hashed
seed

an optional random seed
len

the max limit on the number of characters to hash
Returns:

the hash

Type
number
