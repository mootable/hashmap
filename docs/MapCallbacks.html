For Each Function
A callback to execute on every [key,value] pair of this map iterable.

Example

log the keys and values

const forEachFunction = (value, key) => console.log(key,value)

MatchesPredicate(valueopt, keyopt, iterableopt) → {boolean}

Source:
  • hashmap.js, line 284

Test each element of the map to see if it matches and return

  • true if the key and value match.
  • false if it doesn't.
Examples

Only match keys divisible by 2

const myMatchPredicate = (value, key) => key % 2 === 0;

Only match values which are equal to another key in the map

const myMatchPredicate = (value, key, mapIterable) => mapIterable.has(value);

An alternative implementation, (but potentialy slower, and assumes no undefined value)

const myMatchPredicate = (value, key, mapIterable) => mapIterable.indexOf(key) !== undefined;
Parameters:
Name Type Attributes Description
value * <optional>

the entry value.
key * <optional>

the entry key
iterable MapIterable <optional>

the calling Map Iterable.
Returns:
  • a value that coerces to true if it matches, or to false otherwise.

boolean

Reduce Function
A callback to accumulate values from the Map Iterables [key,value] into a single value.
if initial value is undefined or null, unlike Array.reduce,
no error occurs, and it is imply passed as the accumulator value

Returns:
